# Classic Games Mobile

React Native + Expo mobile app for iOS and Android.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android

# Run on web
npm run web
```
